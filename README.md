Attentiveness Monitoring System using R Studio & Python
📌 Project Overview

The Attentiveness Monitoring System is an AI-powered classroom/student monitoring solution designed to analyze student attentiveness in real-time.
This project uses Python, Machine Learning, Computer Vision, and R Studio for data analysis and visualization.

The system helps monitor:

Student attentiveness
Sleeping detection
Head posture analysis
Real-time monitoring
Report generation and analytics

This project is useful for:

Smart classrooms
Online learning environments
Student engagement analysis
Educational research
🚀 Features
🎥 Real-time face and eye detection
😴 Sleeping/drowsiness detection
📊 Attention analytics using R Studio
🧠 AI-based monitoring system
📸 Image capturing for inattentive students
📝 Hourly report generation
📈 Data visualization and analysis
⚡ Automated monitoring system

📊 Working of the System
The camera captures live video input.
Face and eye detection algorithms analyze the student's posture and eye movement.
The system identifies:
Active students
Inattentive students
Sleeping students
Data is stored for analysis.
R Studio is used to generate:
